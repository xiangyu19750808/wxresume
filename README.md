# wxresume
Quick Start:
1) pnpm i
2) cp apps/api/.env.example apps/api/.env
3) pnpm dev:api
4) curl http://localhost:8080/v1/health
Repo Layout:
- apps/api: Node API
- packages/templates: resume templates
- data: JD dictionary
- samples: sample data
